# GameLibrary WebApi
## Использованные технологии:
### Для тестирования:
- FakeItEasy
- InMemory
- xUnit
### Для WebApi:
- FakeItEasy
- InMemoryDatabase
- xUnit
### В Api реализованы CRUD операции с играми.
- Создание,
- Получение списка игр с возможностью фильтрации по жанру,
- Изменение,
- Удаление.
### Запись об игре содержит данные:
- Название,
- Студия разработчик,
- 1 или более жанров, которым соответствует игра.
